## Requirements

| Dependency | Version |
|---|---|
| Node.js | `>= 18` |
| RxJS | `>= 7.8.0` |
| TypeScript | `>= 5.0` (recommended) |

## Install

<Tabs>
  <TabItem label="npm">
    ```bash
    npm install rxjs-ai rxjs
    ```
  </TabItem>
  <TabItem label="pnpm">
    ```bash
    pnpm add rxjs-ai rxjs
    ```
  </TabItem>
  <TabItem label="yarn">
    ```bash
    yarn add rxjs-ai rxjs
    ```
  </TabItem>
</Tabs>

`rxjs` is a peer dependency — you must install it alongside `rxjs-ai`.

## Package exports

| Import path | What you get |
|---|---|
| `rxjs-ai` | Core primitives, AI controllers, and UI utilities — `createStore`, `createCommandBus`, `createAsyncController`, `createChatController`, `createViewModel` |

```ts
import {
  createStore,
  createCommandBus,
  createAsyncController,
  createChatController,
  createViewModel,
} from 'rxjs-ai';
```

## TypeScript

All types are exported directly from the main entry point. No separate `@types` package is needed.

```ts
import type {
  Store,
  StateUpdater,
  StatePatch,
  CommandBus,
  CommandMap,
  CommandEnvelope,
  AsyncController,
  AsyncState,
  AsyncStatus,
  ChatController,
  ChatModelAdapter,
  ChatModelRequest,
  ChatMessage,
  ChatState,
  ChatStatus,
  ChatRole,
  ChatChunk,
} from 'rxjs-ai';
```

<Aside type="note">
rxjs-ai is written in TypeScript and ships its own type declarations. Enabling `strict` mode in your `tsconfig.json` is recommended for the best experience.
</Aside>
